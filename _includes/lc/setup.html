Git

Git is een version control system waarmee je kunt bijhouden wie wanneer welke veranderingen heeft gemaakt. Je kan er eenvoudig code (of andere documenten) mee delen en updaten op https://github.com.

Volg de instructies in deze les om Git te installeren.

Standaard GitHub accounts zijn gratis. We raden je aan om een Github account aan te maken, als je er niet al één hebt. Overdenk daarbij welke persoonlijke informatie je wilt delen. Lees bijvoorbeeld deze instructies om je emailadres voor jezelf te houden.

Voor Github heb je een ondersteunde browser nodig.
